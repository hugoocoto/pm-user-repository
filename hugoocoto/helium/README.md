# Helium

Third-party desktop client for the [Imput](https://imput.com) browser. Pre-built AppImage (v0.14.5.1). Upstream source at [imputnet/helium-linux](https://github.com/imputnet/helium-linux).

## Install

```lua
ur.Fetch { user = "hugoocoto", file = "helium/helium.lua" }
```

Installs the AppImage and makes it executable. Run `helium` to launch.
