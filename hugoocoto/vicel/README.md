# vicel

Visual Cell Editor: a free TUI spreadsheet editor for the terminal. Edit CSV-like grids with Vim-style motions, formulas, and mouse support. Source at [hugoocoto/vicel](https://github.com/hugoocoto/vicel).

## Install

```lua
ur.Fetch { user = "hugoocoto", file = "vicel/vicel.lua" }
```

## Use

```
vicel [file.csv]
```

Numbers, text, formulas. Vim-style keyboard navigation. Mouse support. Configurable via `.py` file. Lightweight codebase.
