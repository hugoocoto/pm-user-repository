# vicel

Visual Cell Editor: a TUI spreadsheet editor for the terminal. Edit CSV-like grids with Vim-style motions, formulas, and mouse support. Source at [hugoocoto/vicel](https://github.com/hugoocoto/vicel).

![Screenshot](https://raw.githubusercontent.com/hugoocoto/vicel/main/images/image.png)

**Tech stack:** C, ncurses.

**Type:** TUI application.

## Use

```
vicel [file.csv]
```

Numbers, text, formulas. Vim-style keyboard navigation. Mouse support. Configurable via `.py` file.
